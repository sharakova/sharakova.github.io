企業向けメンタルヘルスSaaS

中小企業の健康経営を包括的にサポート

SaaS Laravel Vue.js 健康経営

プロジェクト概要

50人未満の中小企業向けに、ストレスチェックの実施、従業員へのカウンセリング提供、健康経営優良法人認定の取得サポートをワンストップで提供するSaaSプラットフォームを開発しました。

中小企業では、法令で義務付けられたストレスチェックの実施や、従業員のメンタルヘルスケアに関するリソースが不足しています。本サービスは、これらの課題を一括で解決し、企業の健康経営を実現するための包括的なソリューションを提供します。

サービスの特徴

ストレスチェック機能

  • 厚生労働省が推奨する57項目の質問票をベースにしたWeb実施システム
  • 従業員は匿名性を保ちながらオンラインで回答可能
  • 自動集計・分析により、企業全体のストレス状況を可視化
  • 高ストレス者への自動通知とフォローアップ機能
  • 産業医への報告書自動生成

従業員カウンセリング

  • オンラインカウンセリングプラットフォームとの統合
  • 企業が従業員に対してカウンセリング利用枠を提供
  • 利用状況のダッシュボード表示（個人情報は保護）
  • 企業負担の柔軟な設定（全額負担、一部負担など）

健康経営優良法人認定サポート

  • 認定要件のチェックリスト機能
  • 必要な取り組みのステップバイステップガイド
  • 申請書類の自動生成支援
  • 認定取得に向けたコンサルティングサポート

管理者向けダッシュボード

  • 企業全体のメンタルヘルス状況を可視化
  • 部署別・年代別の分析レポート
  • 法令遵守状況の確認
  • カウンセリング利用状況の把握

技術スタック

バックエンド

  • PHP / Laravel: RESTful API、管理画面の実装
  • MySQL: 企業情報、ユーザー情報、ストレスチェック結果の管理
  • Redis: セッション管理とキャッシング
  • Laravel Queue: ストレスチェック集計などのバックグラウンド処理

フロントエンド

  • Vue.js: インタラクティブなダッシュボードUI
  • Chart.js: データ可視化（グラフ・チャート）
  • Tailwind CSS: モダンで保守性の高いスタイリング

インフラ

  • AWS EC2: アプリケーションサーバー
  • AWS RDS: データベース（Multi-AZ構成で冗長性確保）
  • AWS S3: レポートファイルの保管
  • AWS CloudWatch: ログ管理とアラート

主要機能の実装

1. ストレスチェック実施システム

厚生労働省の基準に準拠したストレスチェックを実施するシステムを構築。従業員は匿名でWeb上から回答し、結果は暗号化してデータベースに保存されます。

管理者は、個人が特定できない形で集団分析結果を閲覧可能。高ストレス者には自動で医師面接の案内が送信され、適切なフォローアップが行われる仕組みを実装しました。

2. 自動レポート生成機能

ストレスチェック実施後、自動で以下のレポートを生成：

  • 企業全体の集団分析レポート
  • 部署別・年代別の比較分析
  • 前回実施時との経年比較
  • 産業医向けの報告書

PDFとしてエクスポート可能で、社内資料や産業医への提出資料として活用できます。

3. カウンセリング統合機能

オンラインカウンセリングプラットフォームとAPI連携し、企業管理者が従業員にカウンセリング利用枠を提供できる仕組みを実装。従業員は専用のダッシュボードから簡単にカウンセリングを予約・利用できます。

4. 健康経営認定支援機能

健康経営優良法人認定の要件を満たすための取り組みをステップバイステップでガイド。現在の達成状況を可視化し、不足している項目を明確に表示します。必要な証拠書類のテンプレートも提供し、申請プロセスを大幅に簡素化しました。

セキュリティとコンプライアンス

個人情報保護

  • ストレスチェック結果は暗号化して保存
  • 管理者は個人を特定できない集団分析結果のみ閲覧可能
  • 本人同意なしには結果を開示しない仕組みを実装
  • データは法令で定められた期間適切に管理

法令遵守

  • 労働安全衛生法に準拠したストレスチェック実施
  • 個人情報保護法に準拠したデータ管理
  • 定期的な監査とセキュリティアップデート

開発における課題と解決策

課題1: 匿名性の担保

課題: ストレスチェック結果の匿名性を担保しつつ、高ストレス者へのフォローアップを実現する必要があった

解決策: 結果データと個人情報を分離し、本人のみがアクセスできるトークンベースの仕組みを実装。管理者は統計データのみ閲覧可能とし、個人特定を不可能にしました。

課題2: 複雑な集計ロジック

課題: 部署別・年代別・性別など多軸での分析が必要で、集計処理が複雑

解決策: Laravel Queueを活用したバックグラウンド処理と、適切なデータベースインデックス設計により、大規模なデータでも高速に集計できる仕組みを実現しました。

プロジェクトの成果

  • サービス開始から1年で50社以上が導入
  • 導入企業の健康経営優良法人認定取得率80%
  • 従業員満足度の向上（導入企業平均15%アップ）
  • ストレスチェック実施の工数を従来の1/10に削減
  • 複数の自治体・商工会議所から推奨サービスとして認定

今後の展開

現在は50人未満の企業向けにサービスを提供していますが、今後は中堅企業向けのエンタープライズプランも展開予定です。また、AIを活用したストレス予測機能や、健康経営の改善提案機能なども開発中です。
